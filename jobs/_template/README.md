# Job Folder Template

Create one folder per job under `jobs/` using a clear slug, then place these files inside it:

- `job-description.md`
- `cv-focus.md`
- `cover-letter.md`

Optional extras:

- `statement.md`
- `interview-notes.md`
- `source-files.md`

## Workflow

1. Paste or summarise the JD into `job-description.md`
2. Write the role emphasis in `cv-focus.md`
3. Draft the cover in `cover-letter.md`
4. Pull facts from `Cv/master_capability_tree.md` and `Cv/base_cv_template.md`

Keep all tailoring in the job folder so each application has its own history.
